DDL

SQL — CREATE TABLE

Generado por Hibernate (Spring Boot JPA). La BD crea estas tablas al iniciar si spring.jpa.hibernate.ddl-auto=update o create.

album 
CREATE TABLE album (
  id          BIGINT       NOT NULL AUTO_INCREMENT,
  nombre      VARCHAR(255) NOT NULL,
  year        INT          NOT NULL,
  descripcion VARCHAR(500),
  is_active   BOOLEAN      DEFAULT TRUE,
  created_at  DATETIME(6),
  updated_at  DATETIME(6),
  PRIMARY KEY (id)
);
lamina 
CREATE TABLE lamina (
  id                 BIGINT       NOT NULL AUTO_INCREMENT,
  album_id           BIGINT       NOT NULL,
  nombre             VARCHAR(255) NOT NULL,
  imagen             VARCHAR(255),
  fecha_lanzamiento  DATE         NOT NULL,
  tipo_lamina        VARCHAR(255) NOT NULL,
  is_active          BOOLEAN      DEFAULT TRUE,
  created_at         DATETIME(6),
  updated_at         DATETIME(6),
  PRIMARY KEY (id),
  CONSTRAINT fk_lamina_album FOREIGN KEY (album_id) REFERENCES album(id)
);
lamina_catalogo 
CREATE TABLE lamina_catalogo (
  id                 BIGINT       NOT NULL AUTO_INCREMENT,
  album_id           BIGINT       NOT NULL,
  nombre             VARCHAR(255) NOT NULL,
  imagen             VARCHAR(255),
  fecha_lanzamiento  DATE         NOT NULL,
  tipo_lamina        VARCHAR(255) NOT NULL,
  is_active          BOOLEAN      DEFAULT TRUE,
  created_at         DATETIME(6),
  updated_at         DATETIME(6),
  PRIMARY KEY (id),
  CONSTRAINT uq_catalogo_album_nombre UNIQUE (album_id, nombre),
  CONSTRAINT fk_catalogo_album FOREIGN KEY (album_id) REFERENCES album(id)
);
Reglas

Convenciones de persistencia

SD Soft Delete

Ninguna entidad se elimina físicamente. El campo is_active = false marca el registro como eliminado. Todos los repositorios filtran por findByActiveTrue().

AU Auditoría JPA

created_at y updated_at se gestionan automáticamente con @EntityListeners(AuditingEntityListener.class). Habilitado vía JpaAuditingConfig.

UQ Catálogo único

Un álbum no puede tener dos láminas de catálogo con el mismo nombre. El constraint compuesto (album_id, nombre) en lamina_catalogo lo garantiza a nivel de BD.
