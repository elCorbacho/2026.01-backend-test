Setup rápido

Cómo montar la aplicación

Pasos mínimos para levantar el backend de Álbumes y Láminas en local.

Java 21
Spring Boot 3.5.9
Maven Wrapper

Prerrequisitos

  • Java 21 instalado y en el PATH.
  • Maven (usa el wrapper incluido: mvnw).
  • Opcional: MySQL/MariaDB si quieres conectar a una base propia.
  • Puerto libre 8080 (configurable vía server.port).

Comandos esenciales

Clonar y construir 
git clone <repo-url>
cd 2026.01-backend-test
# Windows
.\mvnw.cmd clean install
# Linux/Mac
./mvnw clean install
Ejecutar la app 
# Windows
.\mvnw.cmd spring-boot:run
# Linux/Mac
./mvnw spring-boot:run

Config rápida (application.properties)

Ajusta credenciales de BD o el puerto si lo necesitas:

spring.datasource.url=jdbc:mysql://localhost:3306/albumes
spring.datasource.username=usuario
spring.datasource.password=clave
server.port=8080
Si no usas BD local, apunta spring.datasource.url a tu instancia remota.

Verificación

  1. Abre /swagger-ui.html para probar endpoints.
  2. Confirma salud en /actuator/health.
  3. Revisa logs: la app debería iniciar en ~2-3s tras el build.

URLs útiles

  • API base: http://localhost:8080
  • Swagger UI: /swagger-ui.html
  • OpenAPI JSON: /v3/api-docs
  • Health: /actuator/health
