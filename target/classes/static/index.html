Pilares y contrato de respuesta
  • Validación con anotaciones Jakarta en cada DTO de entrada.
  • Servicios transaccionales con @Transactional y readOnly=true en lecturas.
  • GlobalExceptionHandler: ResourceNotFoundException, InvalidOperationException.
  • Soft delete con campo active — nunca se borra una fila físicamente.
  • Auditoría JPA automática vía @CreatedDate / @LastModifiedDate.

Todas las respuestas usan ApiResponseDTO<T>:

ApiResponseDTO.java 
{
  "success":   true,
  "message":   "OK",
  "data":      { ... },
  "timestamp": "2026-02-26T12:00:00Z"
}

Verificar

  • Java 21 en el PATH.
  • Maven wrapper (mvnw / mvnw.cmd) incluido en el repo.
  • Puerto 8080 libre (configurable en application.properties).
  • Opcional: MySQL / MariaDB para producción.

bash 
git clone <repo-url>
cd 2026.01-backend-test

# Windows
./mvnw.cmd clean install
./mvnw.cmd spring-boot:run

# Linux / Mac
./mvnw clean install
./mvnw spring-boot:run

Edita src/main/resources/application.properties:

application.properties 
spring.datasource.url=jdbc:mysql://localhost:3306/albumes
spring.datasource.username=usuario
spring.datasource.password=clave
server.port=8080

  • Swagger UI → /swagger-ui.html
  • Health check → /actuator/health
  • OpenAPI JSON → /v3/api-docs
  • Arranque en ~2-3 segundos tras el build.
Base de datos

Estructura de entidades

Esquema relacional generado desde entidades JPA con auditoría y soft delete.

Album

Contexto de colección: título, año y descripción.

PK id nombre * year descripcion active created_at updated_at
Album.java
@Entity
public class Album {
    @Id @GeneratedValue Long id;
    @NotBlank String nombre;
    Integer year;
    String descripcion;
    Boolean active = true;          // soft delete
    @CreatedDate LocalDateTime createdAt;
    @LastModifiedDate LocalDateTime updatedAt;
    @OneToMany(mappedBy = "album", cascade = ALL, fetch = LAZY)
    List<Lamina> laminas;
}
Lamina

Láminas reales del usuario, ligadas a un álbum específico.

PK id FK album_id nombreimagen fecha_lanzamientotipo_lamina activecreated_atupdated_at
Lamina.java
@Entity
public class Lamina {
    @Id @GeneratedValue Long id;
    @ManyToOne(fetch = LAZY)
    @JoinColumn(name = "album_id", nullable = false)
    Album album;
    String nombre;
    String imagen;
    LocalDate fechaLanzamiento;
    String tipoLamina;
    Boolean active = true;
    @CreatedDate LocalDateTime createdAt;
    @LastModifiedDate LocalDateTime updatedAt;
}
LaminaCatalogo

Catálogo maestro de láminas por álbum, con unicidad en nombre dentro del mismo álbum.

PK id FK album_id nombre (unique)imagen fecha_lanzamientotipo_lamina activecreated_atupdated_at
LaminaCatalogo.java
@Entity
@Table(uniqueConstraints = @UniqueConstraint(
    columnNames = {"album_id", "nombre"}))
public class LaminaCatalogo {
    @Id @GeneratedValue Long id;
    @ManyToOne(fetch = LAZY)
    @JoinColumn(name = "album_id", nullable = false)
    Album album;
    @NotBlank String nombre;
    String imagen;
    LocalDate fechaLanzamiento;
    String tipoLamina;
    Boolean active = true;
    @CreatedDate LocalDateTime createdAt;
    @LastModifiedDate LocalDateTime updatedAt;
}
Cancion

Entidad independiente para gestión de canciones con CRUD completo.

PK id titulo *artista * duracion (seg)genero activecreated_atupdated_at
Cancion.java
@Entity
public class Cancion {
    @Id @GeneratedValue Long id;
    @NotBlank String titulo;
    @NotBlank String artista;
    Integer duracion;   // segundos
    String genero;
    Boolean active = true;
    @CreatedDate LocalDateTime createdAt;
    @LastModifiedDate LocalDateTime updatedAt;
}
